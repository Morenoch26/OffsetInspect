# Contributing to OffsetInspect

Contributions are welcome and appreciated. This includes:

- Bug reports  
- Feature requests  
- Code improvements  
- Documentation updates  

Please follow the guidelines below.

---

## How to Contribute

### 1. Fork the repository
Click **Fork** in the upper-right corner.

### 2. Create a feature branch
```powershell
git checkout -b feature/my-improvement
```

### 3. Commit your changes
```powershell
git commit -m "Add: description of improvement"
```

### 4. Push your branch
```powershell
git push origin feature/my-improvement
```

---

### 5. Open a Pull Request
Include:

- Summary of changes
- Reason for the change
- Any relevant testing steps

---

## Coding Guidelines

- Follow PowerShell best practices
- Avoid external dependencies
- Document new functionality clearly
- Ensure features do not break existing behavior
- Include examples when appropriate

---

## Reporting Issues

When submitting a bug report, include:

- PowerShell version
- Operating system version
- Full command used
- Relevant output or error messages

Clear, reproducible issue reports greatly help improve the tool.
